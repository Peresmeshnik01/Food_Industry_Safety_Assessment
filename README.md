Это десктопное приложение, разработанное в рамках расчетно-графической работы (РГР) по дисциплине "Теория и методы программирования". Данная расчётно-графическая работа - игра в жанре quiz,
направленная на обучение нейронной сети классификации изображений по двум классам: "свежий" и "тухлый" продукт. Для обучения модели нейросети я использовала Искусственный Интеллект обучаемого типа модели и метод обучения с учителем (Supervised Learning), который был интегрирован в игровой процесс.
Команда для установки библиотек: pip install Pillow numpy tensorflow keras scikit-learn matplotlib pygame
